回應式設計(RWD)

使用回應式設計可以讓我們製作的網頁隨著螢幕大小或是使用pc的不同(筆電, 手機,平板…)來轉換畫面的呈現.

概念

如果想要改變排版,讓程式偵測到螢幕畫面寬度改變了,就可以讓所有的區塊因為寬度不足而往下掉,形成縱向的排版!

運用技巧

Css Media Query

Meta 標籤

在head標籤底下加入meta標籤,為的是跟瀏覽器說不要任意改變畫面的大小,並且固定縮放的大小
