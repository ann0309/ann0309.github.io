Flex Box 排版

2021-11-02
Coding

讓排版更方便
before: display: inline block

利用Flex Box解決排版難題:

  1. 水平對齊
  2. 並排(切欄)
  3. 垂直對齊

問題1: 並排

Solve:

  1. item(粉色區塊)要設定flex:none, 代表粉色區塊不會隨著頁面的拉動改變大小
  2. 容器(黃色區塊)要設定 display: flex,使得區塊並排
    這兩個都要設定才會有結果!
    未設定前:

設定後:

問題2: 水平對齊

Solve:
容器設定 justify-content: center;
代表讓容器內的東西處在畫面的中間

問題3: 垂直對齊

未設定前:

Solve:
容器裡設定 align-items: center(垂直方向置中)
預設值為stretch

flex屬性:

auto:不用設定寬度, 會根據頁面寬度大小填滿頁面寬度
none:固定寬度, 沒辦法隨著頁面寬度縮放
