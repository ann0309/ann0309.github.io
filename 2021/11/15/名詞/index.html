Internet,Intranet,Extranet 網路

Internet

  • 以TCPIP協定連線全世界的計算機網路的資料通訊網，讓網路使用者共享的資訊資源

(不用通行證)

Intranet

  • 企業內部網路，只提讓企業或部門的人取用網路中的資料通訊網
    採用TCP/IP協議
  • 利用Internet的Web概念與技術,通過防火牆把內部網路和Internet隔開。

(要進去Intranet需要有通行證)

Extranet

把自己機構的Intranet和部分的人分享
(要有別人發給你他們家的通行證)

三者主要以可以取用的人來做區分

Data Mart

資料市場。一種資料倉儲的方式

Meta data

  • 用來描述資料的資料,對數據及訊息資源更詳細的描述訊息。
  • 任何有大量資料的地方都需要metadata

    ETL(Extract-Transform-Load)

  • 描述將資料從來源端經過抽取（extract）、轉置（transform）、載入（load）至目的端的過程。
  • 較常用在資料倉儲，不限於資料倉儲。
  • 愈大量的資料、複雜的轉換邏輯、目的端為較強運算能力的資料庫，愈偏向使用ELT

OLTP 線上交易處理(online transaction processing)

  • 指系統能夠處理大量的更新以及新增的查詢
  • 因為可能同時有多個使用者所以需要確保數據一致性(ACID)
    EX: 銀行系統。確保每次拿取的資料都是一致性且ATM金額是正確的

    OLAP( Online Analytical Processing)

  • 用來做歷史資料的分析以及報告
  • 以讓數據聚合 (data aggregation) 和批次處理 (Batch processing)
  • 可以很快的用query來得到數據並且分析數據聚合後的結果
    EX:商業智慧分析
  • 兩者差在速度和需不需要資料一致性*

API (Application Programming Interface)

  • 使用 API 的過程不需要知道其內部程式運作的邏輯或演算法。只要告訴 API 它需要知道的事，它就會把你想知道的結果帶給你。
  • 應用程式提供的一個工具，讓開發者可以使用 API 來存取該應用程式的資料或是服務等等。
  • 開發者必須按照應用程式 API 的規範才能存取相對應的功能或資料。
  • 有點像是利用api把別人的服務導入到自己的網站。
    EX: 哪天 Google map 將他的 API 關掉，世界上就沒有任何應用程式可以使用 Google map 的功能了。
