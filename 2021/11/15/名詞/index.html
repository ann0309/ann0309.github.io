一些我不知道的名詞

Internet,Intranet,Extranet 網路

Internet

以TCPIP協定連線全世界的計算機網路的資料通訊網，讓網路使用者共享的資訊資源

不用通行證

Intranet

  • 企業內部網路，只提讓企業或部門的人取用網路中的資料通訊網
    採用TCP/IP協議
  • 利用Internet的Web概念與技術,通過防火牆把內部網路和Internet隔開。

要進去Intranet需要有通行證

Extranet

把自己機構的Intranet和部分的人分享
要有別人發給你的他們家的通行證

三者主要以可以取用的人來做區分

Data Mart

資料市場。一種資料倉儲的方式

Meta data

ETL

Extract-Transform-Load
用來描述將資料從來源端經過抽取（extract）、轉置（transform）、載入（load）至目的端的過程。ETL一詞較常用在資料倉儲，但其物件並不限於資料倉儲。
愈大量的資料、複雜的轉換邏輯、目的端為較強運算能力的資料庫，愈偏向使用ELT

OLTP

online transaction processing
線上交易處理

API

Application Programming Interface
使用 API 的過程時，你不需要知道其內部程式運作的邏輯或演算法，你只要告訴 API 它需要知道的事，它就會把你想知道的結果帶來給你。
