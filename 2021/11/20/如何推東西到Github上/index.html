如何推東西到Github上?
2021-11-20 00:00:00

本機端

  1. 找到想要放在Github上的資料夾，進到目錄底下
    cd ..彭彭…

  2. 初始化git repository,在檢視之下把隱藏的項目打勾，會看到多了一個.git的資料夾出現
    git init

  3. 確認Repository狀態，並且查看預設的分支是甚麼(這裡分支是master)
    git status

  4. 當把要推上Github的東西放入當前你所在的資料夾，確認狀態之後，會發現出現很多紅字，代表他發現這些檔案是未知狀態的檔案。

  5. 把檔案加入追蹤清單，並且查看狀態。綠色的字代表他已經更新這些檔案的狀態了
    git add .

  6. 建立更新訊息。此處的版本訊息是 front end
    git commit -m “front End”

  7. 在確認一次狀態會發現她說現在所在的分支是master，並且沒東西可以更新了。這樣就可以在Github上看到你推上去的咚咚了。

將本機與遠端連結

  1. 遠端查詢(如果還沒有連到遠端，就不會跑出東西)
    git remote -v
  2. 這裡我設放檔案的遠端空間叫做Coding，後面的網址為遠端空間的地址。

git reomte add Coding https:…….
3. 確認遠端空間。他會告訴我檔案放的遠端空間名稱和其地址
git remote -v
4. 把剛剛本機所在的分支master推到叫做Coding遠端空間(這裡就想像Coding和master中間有一個=)
git push Coding master
5. 成功!!!!!要記得確認狀態喔~
