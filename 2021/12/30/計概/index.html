計概

Internet,Intranet,Extranet 網路

Internet

  • 以TCPIP協定連線全世界的計算機網路的資料通訊網，讓網路使用者共享的資訊資源

(不用通行證)

Intranet

  • 企業內部網路，只提讓企業或部門的人取用網路中的資料通訊網
    採用TCP/IP協議
  • 利用Internet的Web概念與技術,通過防火牆把內部網路和Internet隔開。

(要進去Intranet需要有通行證)

Extranet

把自己機構的Intranet和部分的人分享
(要有別人發給你他們家的通行證)

三者主要以可以取用的人來做區分

RAM,ROM,硬碟

Read access, Read only

| RAM | ROM |
| ——– | ——– |
| 內存 | 存儲 |
| 隨機存取 | 唯讀 |
